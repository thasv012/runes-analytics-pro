Como Implementar

Adicione o sistema de compartilhamento à sua aplicação RUNES Analytics Pro com apenas algumas linhas de código: 

// 1. Inclua os arquivos necessários no HTML
<link rel="stylesheet" href="css/share-tools.css">
<script src="services/sharing/SharingService.js"></script>
<script src="components/ShareTools.js"></script>

// 2. Adicione um elemento container
<div id="sharing-tools-container"></div>

// 3. Inicialize o componente com JavaScript
const sharingTools = new ShareTools('sharing-tools-container', {
    enabledFeatures: {
        shareUrl: true,
        socialCards: true,
        dataExport: true
    },
    defaultFileName: 'runes-data-export'
});

// 4. Configure os dados a serem compartilhados
sharingTools.setData(
    {
        title: 'Análise do token PEPE',
        description: 'Estatísticas e métricas para o token PEPE',
        statLabel: 'Holders Ativos',
        statValue: '2.483',
        holders: '2.4K',
        transactions: '18.5K'
    },
    {
        view: 'token-details',
        token: 'PEPE',
        period: '30d'
    }
);

Configurações Avançadas

O sistema de compartilhamento do RUNES Analytics Pro inclui diversas opções avançadas de configuração: 

// Configuração avançada da compressão de estado
sharingService.configure({
    compressionEnabled: true,
    shortUrlEnabled: true,
    shortUrlService: 'bitly',
    bitlyToken: 'seu_token_bitly',
    securitySettings: {
        expirationTime: 7 * 24 * 60 * 60 * 1000, // 7 dias
        maxStateSize: 200 * 1024, // 200KB
        sensitiveFields: ['apiKey', 'userToken', 'password']
    }
});

// Personalização de templates para cartões sociais
sharingService.configure({
    socialCardTemplates: {
        twitter: {
            width: 1200,
            height: 628,
            template: 'meu-template-twitter' // ID do elemento HTML com template
        }
    }
});

Sobrescrevendo o Método getCurrentState

Para capturar o estado atual da aplicação de forma personalizada: 

// Subclasse com estado personalizado
class RunesShareTools extends ShareTools {
    getCurrentState() {
        // Capturar estado de filtros, visualizações, etc.
        const activeFilters = runExplorer.getActiveFilters();
        const sortSettings = runExplorer.getSortSettings();
        const selectedToken = runExplorer.getSelectedToken();
        
        return {
            filters: activeFilters,
            sort: sortSettings,
            token: selectedToken,
            view: currentView,
            timestamp: new Date().toISOString()
        };
    }
}
